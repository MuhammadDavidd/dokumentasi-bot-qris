📘 Dokumentasi Bot QRIS Totoro Store

Pengembang: Muhammad David Aryanto
Gateway: ORDERKOUTA
Bot Discord: Otomatisasi sistem pembayaran via QRIS

🔧 Fitur Utama

  • /bayar - Membuat QRIS pembayaran otomatis
  • /statistik - Menampilkan statistik transaksi (total, berhasil, pendapatan, hari ini)
  • /cetak-bukti - Cetak bukti pembayaran dalam bentuk gambar
  • /buat-laporan - Menghasilkan laporan transaksi bulanan dalam format Excel (khusus Owner)
  • /uptime - Menampilkan status uptime bot
  • /info - Menampilkan informasi tentang bot dan perintahnya
  • /set-info - Menentukan channel untuk notifikasi transaksi (khusus Owner)
  • /set-nota - Menentukan channel untuk nota otomatis (khusus Owner)

📤 Notifikasi Otomatis

  • Transaksi baru akan dikirim ke channel info
  • Transaksi berhasil akan dikirim ke channel info
  • Nota pembayaran otomatis dikirim ke channel nota

📄 File & Struktur Data

  • qris_data.json - Menyimpan semua data transaksi
  • channels_config.json - Menyimpan pengaturan channel info/nota
  • qr_images/ - Folder menyimpan QR code
  • reports/ - Folder menyimpan laporan Excel

🛠️ Teknologi

  • discord.js - Untuk interaksi bot Discord
  • qris-payment - API untuk generate & cek QRIS
  • canvas - Untuk membuat bukti pembayaran sebagai gambar
  • exceljs - Untuk membuat laporan transaksi Excel

📌 Catatan

Bot ini akan otomatis memantau status pembayaran hingga 10 menit. Jika pembayaran diterima, akan langsung memproses transaksi dan mengirimkan bukti pembayaran ke user dan channel yang ditentukan.
